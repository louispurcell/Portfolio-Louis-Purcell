Project Description

Project Title: Lunar Outpost – Game Environment Design

Unit: Introduction to Video Games

Software: Unity

In this project for Introduction to Video Games, I was tasked with designing a fully realized 3D game environment using Unity. The focus was on learning key elements of game world creation, including terrain sculpting, lighting, textures, physics, and environmental design.

I created a moon-based map featuring a space outpost, using custom terrain tools to simulate a cratered lunar surface. I sourced and integrated various assets from the Unity Asset Store to populate the map with futuristic structures, machinery, and space-themed props, creating an immersive sci-fi setting.

To enhance realism, I adjusted global lighting to mimic the harsh shadows and cold light of the moon’s surface. I also altered Unity’s gravity settings to replicate low-gravity physics, giving the scene a believable outer-space feel. Special effects such as dust particles and ambient audio helped to further establish the atmosphere.

This project strengthened my skills in Unity’s environment tools and introduced me to essential game design concepts like world-building, atmosphere, and player immersion.

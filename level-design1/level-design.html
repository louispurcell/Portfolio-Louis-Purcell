Project Description

Project Title: 2D Game Level Design

Unit: Game Design

Software: Unity

This project was completed as part of the Game Design unit, where the task was to create a level for a 2D game using Unity. Provided with a set of assets, I was challenged to design a level that thoughtfully incorporates various core game design elements including drama, challenge, exploration, and gameplay mechanics.

The goal was to use the available resources creatively to build an engaging and balanced experience that guides players through a compelling environment. The level was crafted to evoke emotion, present meaningful obstacles, and encourage players to explore, all while integrating core mechanics to support smooth and enjoyable gameplay.
